# Fortex-Website
Fortex company, which sells building materials

Project Overview
This project is a responsive, single-page website developed for Fortex, a company specializing in construction and renovation services. The goal is to create a modern digital presence that allows users to explore the company's services, view a gallery of work, and access contact information seamlessly across all devices.

What We Are Creating

We are building landing page that functions as a digital brochure. The site will feature:

Interactive Navigation: A side menu system for intuitive navigation between sections.
Service Showcase: A detailed breakdown of construction expertise including masonry, roofing, plumbing, and technical installations.
Dynamic Gallery: A portfolio section that loads project details dynamically for a smoother user experience.
Geolocation Integration: A custom-styled map to help clients find the company's physical location.
